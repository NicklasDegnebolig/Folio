# Welcome to folio

A Vite-native MDX content layer. Full documentation is on its way.

<InfoBox>
  You're reading a `.md` file rendered live by the folio Vite plugin — no build
  step, no codegen, just Vite.
</InfoBox>

<InfoBox type="tip">
  folio is framework-agnostic. Swap `jsxImportSource` to target React, Solid, or
  any other JSX-compatible framework.
</InfoBox>

<InfoBox type="warning">
  folio is in early development. The API may change before a stable release.
</InfoBox>
